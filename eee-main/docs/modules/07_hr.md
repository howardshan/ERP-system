# 模块 07：HR（人力资源）

包含：员工目录、员工档案查看与编辑

---

## 模块入口

`activeModule = 'hr'` → `src/pages/hr/HRModule.tsx`

**UI 主题色**: Teal（青色）

**返回首页**: 页面顶部左上角 "← All Modules" 按钮，调用 `onNavigate('home')`

---

## 权限要求

| 权限 | 说明 |
|------|------|
| `hr.employees.view` | 查看员工列表及档案 |
| `hr.employees.edit` | 编辑员工档案（姓名、职位、部门、上级） |

无 `hr.employees.view` 时，模块主体显示权限拒绝画面。

---

## 1. 员工列表（Employee Directory）

**默认视图**，进入 HR 模块后首先展示。

### 列字段

| 列 | 数据来源 |
|----|---------|
| Name（姓名） | `erp_user.full_name` |
| Role（职位） | `erp_user.role` |
| Department（部门） | `erp_user.department` |
| Manager（上级） | `erp_user.manager`（上级姓名） |
| Email（邮箱） | `auth.users.email`（来自 `list_erp_users()` RPC） |

### 操作
- 点击任意行 → 进入员工档案详情页（需 `hr.employees.view`）

### 数据来源
调用 `list_erp_users()` RPC（SECURITY DEFINER），返回 `erp_user LEFT JOIN auth.users` 合并视图，包含 `role` 字段（M-015 新增）。

---

## 2. 员工档案详情（Employee Profile）

**路由**: 点击列表行进入，内部 state 切换视图

**导航**: 顶部左侧 "← All Employees" 按钮，返回员工列表

### 可编辑字段

| 字段 | 对应 DB 列 | 说明 |
|------|------------|------|
| Full Name（全名） | `erp_user.full_name` | 员工显示姓名 |
| Role（职位/职称） | `erp_user.role` | 职位/职称，M-015 新增列 |
| Department（部门） | `erp_user.department` | 所属部门 |
| Manager（上级） | `erp_user.manager_id` | 选择上级领导（下拉，来自 `erp_user` 列表） |

> **注意**：档案编辑从 `UserDetail`（Auth 模块）迁移至此模块，Auth 模块的 `UserDetail` 不再包含 Profile Information 卡片。

### 权限控制

| 元素 | 所需权限 |
|------|---------|
| 查看档案 | `hr.employees.view` |
| Edit / Save 按钮 | `hr.employees.edit` |
| 字段可编辑状态 | `hr.employees.edit`（否则字段只读） |

### 保存机制

调用 `src/services/authApi.ts` 中的 `updateUser(userId, { full_name, role, department, manager_id })`，直接 UPDATE `erp_user` 表对应记录。

---

## 数据库层

| 表 / 函数 | 说明 |
|-----------|------|
| `erp_user` | ERP 用户档案主表，含 `role` 字段（M-015 新增） |
| `list_erp_users()` | SECURITY DEFINER RPC，返回包含 `role` 的合并视图 |
| `updateUser()` | `src/services/authApi.ts`，UPDATE `erp_user` 指定字段 |

---

## 权限结构

```typescript
hr: {
  label: 'HR',
  resources: {
    module_permissions: { permissions: [manage] },
    employees: {
      label: 'Employees',
      permissions: [
        { id: 'view', label: 'View Employees', prereq: null },
        { id: 'edit', label: 'Edit Employee Profile', prereq: 'view' },
      ]
    },
    audit_log: {
      label: 'Audit Log',
      permissions: [{ id: 'manage', label: 'Manage HR Audit Log', prereq: null }]
    }
  }
}
```
