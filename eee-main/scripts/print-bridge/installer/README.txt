ERP 标签打印助手 — 安装说明
================================

这个小程序让浏览器里的 ERP 系统可以静默打印标签（无需弹打印对话框），
打印到本机连接的标签打印机。纸张尺寸：4 × 3 英寸（横版）。

────────────────────────────────────────────────────────────
macOS
────────────────────────────────────────────────────────────
1. 双击  install-macos.command
2. 若提示「无法验证开发者」：右键点该文件 → 打开 → 再点「打开」放行
   （本程序未做签名，属正常一次性提示）
3. 看到「✓ 打印助手已安装并启动」即完成
4. 打开「系统设置 - 打印机与扫描仪」，把标签机默认纸张设为 4×3 英寸
5. 打开 ERP，点右上角打印机设置 → 搜索 CUPS 打印机 → 选中本机打印机

卸载：双击 uninstall-macos.command

────────────────────────────────────────────────────────────
Windows
────────────────────────────────────────────────────────────
1. 双击  install-windows.bat
2. 若 SmartScreen 弹出：点「更多信息」→「仍要运行」
3. 看到「✓ 打印助手已安装并启动」即完成
4. 打开「设置 - 打印机和扫描仪」，把标签机默认纸张设为 4×3 英寸
5. 打开 ERP，点右上角打印机设置 → 搜索 CUPS 打印机 → 选中本机打印机

卸载：双击 uninstall-windows.bat

────────────────────────────────────────────────────────────
每台打印机的一次性设置（型号无关，任何热敏标签机通用）
────────────────────────────────────────────────────────────
不同型号的机器，代码侧无需改动；只要在每台机器上把它连接的打印机
做好下面三件事一次即可：

1. 默认纸张设为 4×3 英寸（系统「打印机与扫描仪」里这台打印机的默认值）。
2. 做一次「自动校准 / 学习标签长度」（Auto-Calibration / Media Learn，
   一般是按住机器走纸键开机或在驱动里点校准）。不校准的话，部分机器会
   每隔几张就重新找间隙，卡十几秒——这正是"打印很慢/间歇停顿"的原因。
3. 浓度（Darkness/Density）调低一两档、速度（Speed）适当调慢：浓度过高
   会热渗、字边发毛；这是纯硬件设置，不同机器各调各的。
4. 在 ERP 打印机设置里把「打印机分辨率」选成跟这台机器一致（203 / 300 /
   600 dpi，多数热敏机是 203）。选对了标签按原生点阵 1:1 输出，最锐；选错
   会被缩放、边缘发毛。

────────────────────────────────────────────────────────────
说明
────────────────────────────────────────────────────────────
- 打印助手安装后会在后台常驻、开机自启，监听本机 127.0.0.1:6543。
- 每台电脑各自安装、各自选择自己连接的打印机，互不影响。
- 标签按打印机原生 203 dpi 渲染、1:1 输出，不缩放、不绑型号。
- 不用安装 Node 或任何其它环境，程序是自包含的。
